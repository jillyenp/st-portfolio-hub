⚙️ One-Time Setup Required

To enable cloud sync so everyone on the team sees the same links, follow these 3 steps:

1
Go to jsonbin.io and create a free account. You get 10,000 requests/month — more than enough.
2
Create a new Bin and paste the contents of the INITIAL_LINKS array (printed in the browser console on first load). Copy the Bin ID from the URL.
3
Open this HTML file, find the CONFIG section at the top of the <script> block, and paste your API Key and Bin ID:
const CONFIG = {
  JSONBIN_API_KEY: "$2a$10$z9d8UhSlMnB/pG9UTTiH2eakObSbu6YTu.wAyMbefxe.kDUjaJgWe",
  JSONBIN_BIN_ID:  "69df1b03aaba882197fe4a13",
};

Once configured, reload the page. The banner disappears and cloud sync starts automatically. 🎉
